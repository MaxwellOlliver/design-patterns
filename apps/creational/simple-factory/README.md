# simple-factory

A creational design pattern implementation in TypeScript.

## Getting Started

```bash
# Install dependencies
pnpm install

# Run in development mode (with hot reload)
pnpm dev

# Build for production
pnpm build

# Run production build
pnpm start

# Type check
pnpm type-check

# Clean build artifacts
pnpm clean
```
